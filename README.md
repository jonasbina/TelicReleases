# Telic - Alternative Downloads

This repository hosts official, direct downloads for **Telic**, the all-in-one productivity app.
For more info about Telic, visit https://telic.dev

### 🛠️ Installation Note
Because these files are distributed outside of the Play Store/App Store, your device might show a "Unknown Sources" or "Unrecognized Developer" warning. This is normal for direct downloads.
